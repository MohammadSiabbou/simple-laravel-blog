## Test Project 

Technologies Used :
    - Vuejs
    - inertiajs , DOCS : [inertiajs](https://inertiajs.com)
    - Laravel
    - jetstream , DOCS : [jetstream](https://jetstream.laravel.com)
    - MySql
 
 Get Started :
    1 - composer update
    3 - npm install
    4 - npm run dev
    5 - php artisan migrate
 To Serve The App Run : npm run dev & php artisen serve at the same time
